# 紫菜不菜·Photobooth 相框模板网站

## 本版模板

本版共 22 款模板，已统一重新编号、重新命名，并将图片文件统一为 `template-zc-XX.png`。

## 后续新增模板

将新图片放入 `assets` 文件夹，并在 `app.js` 顶部的 `templates` 数组继续增加编号即可。网页总数量现在会自动计算，无需再手动修改首页数量。

## 水印

当前沿用斜向重复的 ZICOI PHOTOBOOTH 水印，透明度为 25%。
